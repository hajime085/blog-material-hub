📥 data.json をインポート

リポジトリにある既存の data.json を読み込んで、作業リストに追加します。新規追加の場合は不要です。

📤 data.json をエクスポート

作業リストの内容を data.json 形式でダウンロードします。ダウンロード後、リポジトリの data.json に上書きして git push してください。

🔍 ローカルプレビュー用に保存

ファイルを直接ブラウザで開いてギャラリーを確認したい場合に使用します。作業リストの内容を localStorage に保存し、gallery.html でプレビューできます。
※ git push 後の本番確認には不要です。

🗑️ 作業リストをリセット

ブラウザ上の作業リストを全件削除します。エクスポート済みの data.json には影響しません。

📖 運用フロー

基本の流れ
  1. PC で WebP 画像ファイルを用意する（例：drink_001.webp）
  2. ファイルを assets/materials/{大分類}/ フォルダに配置
  3. 「新規登録」タブで素材情報を入力して「作業リストに追加」
  4. 「データ管理」→「data.json をダウンロード」
  5. リポジトリの data.json を上書き保存
  6. git add . → git commit → git push
  7. ギャラリーに反映されることを確認 🎉
